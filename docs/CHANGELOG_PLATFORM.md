## 2026-07-02 ‚ÄĒ MM12.3 ‚ÄĒ GuardedSignalSource (runtime boundary guard)

Implemented `core/runtime/guarded_signal_source.py`: the platform-owned `SignalSource` decorator that enforces the certified Strategy Contract at runtime (ADR-018 reject-and-journal boundary validation; ADR-019 quarantine-and-continue fault policy ‚ÄĒ combined in one component per the Technical Lead's non-blocking note). Per bar: short-circuits `[]` once quarantined; runs the inner source inside a fault boundary; validates return shape, timestamp discipline, and mandatory entry risk metadata (`sl_distance`/`risk_r` > 0 on BUY/SELL); drops contract violators and journals `SIGNAL_CONTRACT_REJECTED` while clean siblings still route; quarantines on an uncaught fault or malformed return (journals `STRATEGY_ERROR` + `STRATEGY_QUARANTINED`, edge-triggered, alerts critical, terminal ‚ÄĒ no retry, no auto-flatten). `on_start` faults are a startup refusal (journal, re-raise); `on_stop` faults are logged and swallowed.
